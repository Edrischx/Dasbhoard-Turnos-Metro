# Dashboard Planificación de Turnos

App Streamlit funcional para visualizar planificación de turnos desde Excel.

## Archivos
- `app.py`: aplicación principal
- `base_powerbi_gestion_turnos_metro.xlsx`: datos base
- `requirements.txt`: dependencias

## Ejecutar local
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Publicar en Streamlit Community Cloud
1. Crear repositorio en GitHub.
2. Subir estos archivos.
3. Entrar a https://share.streamlit.io o https://streamlit.io/cloud.
4. Crear app desde el repo.
5. Main file path: `app.py`.
